Compute the upper bound on time before two shapes penetrate. Time is represented as a fraction between \mbox{[}0,t\+Max\mbox{]}. This uses a swept separating axis and may miss some intermediate, non-\/tunneling collision. If you change the time interval, you should call this function again. Note\+: use b2\+Distance to compute the contact point and normal at the time of impact.
